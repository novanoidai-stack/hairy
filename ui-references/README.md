Repositorios de componentes UI premium, sistemas de diseño y arquitecturas de interfaces para referencia. Usar estos recursos para crear diseños que no se vean genéricos.

## 📚 Cómo usar estos recursos

### Para inspiración de componentes:
1. Revisa el componente específico que necesitas (buttons, cards, forms)
2. Mira cómo lo implementan las bibliotecas premium
3. Adapta el patrón al estilo de Mecha (colores fuego, tipografía)

### Para arquitectura de interfaces:
1. Revisa cómo estructuran layouts complejos
2. Estudia los patrones de navegación y jerarquía
3. Adapta las mejores prácticas a tu contexto

### Para motion y animaciones:
1. Revisa las transiciones en NextUI y Chakra
2. Estudia los micro-interactions en Mantine
3. Aplica patrones similares con valores propios

## 🎯 Especificaciones de Mecha

Al usar estas referencias, mantener:
- **Colores**: Acento fuego (#f4501e), fondos crema (#f6f1ea)
- **Tipografía**: Bricolage Grotesque (headings), Inter (body)
- **Motion**: Suave, elegante, no brusco
- **Voz**: Profesional pero cercano, sector peluquería

## 📂 Estructura de directorios

Cada repositorio tiene su propia estructura:

- `/themes` - Sistemas de temas
- `/docs` - Documentación y ejemplos
- `/stories` - Storybook o ejemplos visuales
